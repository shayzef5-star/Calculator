# 🧮 Веб-калькулятор

Простой и стильный веб-калькулятор для выполнения базовых арифметических операций. Сделан с акцентом на удобный интерфейс и адаптивный дизайн.

## 🚀 Демо-версия

[Посмотреть работающий калькулятор](https://shayzef5-star.github.io/Calculator)

## 📸 Preview

![Screenshot of the Calculator](https://shayzef5-star.github.io/Calculator/Preview.jpg)

## 🛠️ Используемые технологии

*   HTML5
*   CSS3
*   JavaScript (ES6+)

## ✨ Возможности

*   Базовые арифметические операции: сложение, вычитание, умножение, деление.
*   Адаптивный интерфейс, корректно отображается на различных устройствах.
*   Чистый и минималистичный дизайн.
