# 🧮 Web Calculator

A simple and stylish web calculator for performing basic arithmetic operations. Designed with a focus on user-friendly interface and responsive layout.

[View the live calculator](https://shayzef5-star.github.io/Calculator)

## 🛠️ Technologies Used

*   HTML5
*   CSS3

*   Basic arithmetic operations: addition, subtraction, multiplication, division.

*   Clean and minimalistic design.
